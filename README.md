# Dependencies Cleaner

> Ferramenta para remover arquivos específicos e classes dentro de arquivos JAR

## 🚀 Tecnologias

- **Rust**
- **Cargo** - Gerenciador de pacotes e compilação

## 📦 Build

Para compilar o projeto, execute o comando na raiz principal:

```bash
cargo build --release
```
